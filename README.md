# Data / Extra_Utilities / Extract_Signer_Specific_Data

This folder contains code for extracting signer-specific data from the complete How2Sign dataset.

## Files

- **image_count.py**
  
  Takes a folder of a specific signer from the classified How2Sign dataset and generates a CSV listing the video names for that signer.

- **video.py**
  
  Takes the CSV generated by `image_count.py` and the complete How2Sign dataset as input, then copies only the videos listed in the CSV to a new folder.
