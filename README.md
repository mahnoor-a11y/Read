# Data / GANs / preprocessing / GAns_dataset_prepration

This folder contain code files for generating GANs training dataset.

## Contents

- **Keypoints_normalization.py**  
  Normalizes keypoints from a folder of JSON keypoint files.

- **draw_skeleton_videos.py**  
  Generates skeleton videos using the normalized keypoints.

- **frame_extraction.py**  
  Extracts frames from both skeleton videos and corresponding human videos.

- **merge_skeleton_and_human_video_frames.py**  
  Concatenates extracted skeleton and human frames side by side.
